Das Projekt liegt schon fertig kompiliert im Release Ordner.

Nach dem kompilieren müssen:

-sfml-graphics-2.dll
-sfml-system-2.dll
-sfml-window-2.dll
im Ordner Release, bzw.

-sfml-graphics-2-d.dll
-sfml-system-2-d.dll
-sfml-window-2-d.dll
im Ordner Debug eingefügt werden.

Die Binaries sind in SFML-2.5.0/bin auffindbar.
